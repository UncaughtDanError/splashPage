A Kubernetes Visualization and Monitoring Application

What is Kubermetrics?

Kubermetrics is an open-source dev tool that provides Kubernetes cluster monitoring as well as data visualization in a simple and easy to understand user interface

What does Kubermetrics solve?

Kubermetrics aims to simplify the user experience while using Kubernetes. As powerful of a tool Kubernetes is, it is not all encompassing and trying to understand what is actually happening inside your cluster can be quite confusing. This is further exemplified when developers start working with clusters that can contain hundreds if not thousands of nodes with a multitude of pods in each one. Thankfully Kubermetrics has the ability to help developers break down the monstrosity of K8s.
